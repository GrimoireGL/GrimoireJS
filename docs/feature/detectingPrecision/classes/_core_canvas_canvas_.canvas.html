Class Canvas

The class to manage HTMLCanvasElement. Provides most of interfaces related to GLContext except the features resource manager providing.

HTMLCanvasElementを管理するクラスs リソースマネージャーが提供する機能以外のGLContextが関連する機能のほとんどを内包します。
